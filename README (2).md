# Part 3: Smokey’s Smoked Chicken and Beef Website

This is the final part of my Web Development assignment, showcasing a fully functional website for **Smokey’s Smoked Chicken and Beef**, a local business based in Polokwane.

---

##  Features Implemented

### Part 2 Fixes
- Fixed folder structure: moved images and CSS into correct subfolders
- Updated all file paths for images and stylesheets
- Cleaned up `index.html` and validated HTML using W3C Validator

###  Part 3 Additions
- Added JavaScript interactivity:
  - Contact form validation
  - Enquiry form with dynamic response messages
- Built and styled `enquiry.html` form with custom logic
- Applied SEO best practices:
  - Meta tags for description, keywords, author
  - Proper heading structure (`<h1>`, `<h2>`, etc.)
  - Descriptive `alt` text for all images
  - Internal linking between pages
- Created `robots.txt` and `sitemap.xml` for search engine indexing
